Lemma V3_determined : V3 = compute_v3 `o [% V1, U1, U2, U3, S, V2].

Description

V3 is functionally determined by the other variables via the constraint. Given V1, U1, U2, U3, S, V2, we can compute: V3 = (S - U2*V2 - U1*V1) / U3. This is key for showing H(V2,V3|constraint) = H(V2|constraint).
