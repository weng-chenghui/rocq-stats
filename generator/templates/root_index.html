About Rocq Stats

Rocq Stats is a documentation generator for Rocq/Coq formalizations. It parses Coq source files to extract lemma information, analyze dependencies, and generate browsable documentation with statistics.

Features

  • Automatic lemma extraction from .v files
  • Main/Helper classification based on declaration type and comments
  • Dependency analysis between lemmas
  • Individual lemma detail pages with source code links
  • Dark/light theme support
